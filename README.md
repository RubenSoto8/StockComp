# StockComp
StockComp es un inventario para gestionar componentes electrónicos. El usuario puede añadir y modificar la cantidad de componentes conforme a los consigue o los utiliza en sus proyectos. Además, la aplicación cuenta con la posibilidad de dividir los componentes en diferentes categorías para una mejor organización.

Al añadir un componente tienes la capacidad de añadir nombre, la categoría a la que pertenece, la cantidad que ingresarás y opcionalmente, es posible agregar notas si se necesitan
